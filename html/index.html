How to publish a blog using AWS

AWS can guarantee amazing speeds for serving a blog - it supports HTTP2, IPv6, caching and has free TLS certificates. Check out how I do it for this blog.
How it looks

The whole process of delivering the content of my blog is based on four steps:

  1. create the content (markdown)
  2. build and publish the content (send generated HTML to somewhere)
  3. serve the build result (accept connections and serve HTML)
  4. invalidating old objects in the edge caches
